This project takes input from the user (city,month,day) and filter the data by this input,
to display the content of the data in order of the most common time of use ,
the most common start and end station , the total time of the usage of all bike,
the average trip time and general statics about the user wethier he is male/female and subscriber/customer.



The project was done the help of python and pandas documentation as well as question posted on stack overflow.
